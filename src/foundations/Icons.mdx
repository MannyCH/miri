# Icons

Miri uses **Feather Icons** - a collection of beautifully simple, open-source icons designed with a focus on simplicity, consistency, and flexibility.

üîó **Icon Library:** [Feather Icons](https://feathericons.com/)

## Why Feather Icons?

‚úÖ **Simple & Clean** - Minimalist 24x24 pixel icons  
‚úÖ **Consistent** - Uniform stroke width and style  
‚úÖ **Lightweight** - Small file sizes  
‚úÖ **Flexible** - Easily customizable with CSS  
‚úÖ **Open Source** - MIT license

## Icon System

### Icon Specifications

- **Size:** 24x24px base size
- **Stroke Width:** 2px
- **Style:** Outlined, minimalist
- **Color:** Inherits from parent or uses color tokens

### Available Icons

Feather Icons includes 280+ icons covering:
- **UI Actions:** Check, X, Plus, Minus, Edit, Trash, Save
- **Navigation:** Chevrons, Arrows, Menu, Home, Settings
- **Communication:** Mail, Message, Phone, Bell
- **Media:** Image, Video, Camera, Music, Film
- **Files:** File, Folder, Download, Upload
- **E-commerce:** Shopping Cart, Credit Card, Tag
- **And more...**

Browse the full collection at [feathericons.com](https://feathericons.com/)

## Installation & Usage

### Option 1: NPM Package (Recommended for React)

```bash
npm install feather-icons
# or
npm install react-feather
```

#### With React

```jsx
import { Calendar, Check, X, ChevronRight } from 'react-feather';

function MyComponent() {
  return (
    <div>
      <Calendar size={24} color="var(--color-icon-neutral)" />
      <Check size={20} color="var(--color-icon-success)" />
      <X size={20} color="var(--color-icon-error)" />
      <ChevronRight size={16} color="var(--color-icon-brand)" />
    </div>
  );
}
```

### Option 2: SVG Sprites

```html
<!-- Include sprite -->
<script src="https://unpkg.com/feather-icons"></script>

<!-- Use icon -->
<i data-feather="calendar"></i>
<i data-feather="check"></i>

<script>
  feather.replace();
</script>
```

### Option 3: Inline SVG

```jsx
<svg 
  width="24" 
  height="24" 
  viewBox="0 0 24 24" 
  fill="none" 
  stroke="currentColor" 
  strokeWidth="2"
>
  <path d="M9 11l3 3L22 4"/>
  <path d="M21 12v7a2 2 0 01-2 2H5a2 2 0 01-2-2V5a2 2 0 012-2h11"/>
</svg>
```

## Icon Sizes

Feather Icons are designed at 24x24px but can be scaled:

| Size | Use Case | Example |
|------|----------|---------|
| **16px** | Small buttons, dense lists | Tags, metadata |
| **20px** | Standard buttons, navigation | Navigation items, actions |
| **24px** | Default size, content icons | Recipe cards, headers |
| **32px** | Large touch targets | Mobile navigation, CTAs |
| **48px+** | Hero sections, empty states | Feature illustrations |

```jsx
// Small (16px)
<CalendarIcon size={16} />

// Standard (20px)
<CalendarIcon size={20} />

// Default (24px)
<CalendarIcon size={24} />

// Large (32px)
<CalendarIcon size={32} />
```

## Best Practices

‚úÖ **DO:** Use semantic color tokens with icons  
‚úÖ **DO:** Maintain consistent icon sizes within a component  
‚úÖ **DO:** Add descriptive `aria-label` for icon-only buttons  
‚úÖ **DO:** Use `currentColor` for flexible icon colors  

‚ùå **DON'T:** Mix icon styles (outlined vs filled)  
‚ùå **DON'T:** Use icons smaller than 16px  
‚ùå **DON'T:** Forget accessibility labels  
‚ùå **DON'T:** Overuse icons (text is often clearer)

## Accessibility

### Icon-Only Buttons

Always provide text alternatives for icon-only buttons:

```jsx
// Good - With aria-label
<button aria-label="Delete recipe">
  <TrashIcon size={20} color="var(--color-icon-error)" />
</button>

// Better - With visible text
<button>
  <TrashIcon size={20} color="var(--color-icon-error)" />
  <span>Delete</span>
</button>

// Best - With visually hidden text
<button>
  <TrashIcon size={20} color="var(--color-icon-error)" />
  <span className="sr-only">Delete recipe</span>
</button>
```

### Decorative Icons

Mark purely decorative icons with `aria-hidden`:

```jsx
<div>
  <CalendarIcon size={16} aria-hidden="true" />
  <span>February 10, 2026</span>
</div>
```
