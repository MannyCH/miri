# Design Foundations

Welcome to the Miri design system foundations. These are the core building blocks that define the visual language of the Miri meal planning application.

## What are Design Foundations?

Design foundations (also called design tokens) are the smallest pieces of our design system. They define:

- **Colors** - Semantic colors for text, backgrounds, strokes, fills, and icons
- **Spacing** - Consistent spacing values for layout and components
- **Typography** - Text styles for headings, body text, and UI elements
- **Border Radius** - Corner radius values for component edges

## Why Design Tokens?

Design tokens provide:

‚úÖ **Consistency** - Use the same values across all components  
‚úÖ **Maintainability** - Update once, change everywhere  
‚úÖ **Accessibility** - Semantic names ensure proper color contrast  
‚úÖ **Themability** - Easy to add dark mode or alternative themes  

## Token Structure

All tokens are exported as CSS custom properties (CSS variables) from Figma and follow this naming convention:

```css
/* Format: --category-subcategory-variant */
--color-text-strong: #260B00F2;
--spacing-16: 16px;
--corner-radius-8: 8px;
```

## Using Tokens in Code

### With Base UI Components

```jsx
import { Button } from '@base-ui/react/button';

// Using inline styles
<Button.Root 
  style={{
    backgroundColor: 'var(--color-fill-brand-strong)',
    padding: 'var(--spacing-16)',
    borderRadius: 'var(--corner-radius-8)',
  }}
>
  Click me
</Button.Root>

// Using CSS classes
<Button.Root className="button-primary">
  Click me
</Button.Root>
```

```css
/* In your CSS file */
.button-primary {
  background-color: var(--color-fill-brand-strong);
  padding: var(--spacing-16);
  border-radius: var(--corner-radius-8);
}
```

### With Plain HTML/CSS

```html
<div class="card">
  <h2 class="card-title">Recipe Card</h2>
</div>
```

```css
.card {
  background: var(--color-background-raised);
  padding: var(--spacing-24);
  border-radius: var(--corner-radius-16);
  border: 1px solid var(--color-stroke-weak);
}

.card-title {
  color: var(--color-text-strong);
  margin-bottom: var(--spacing-12);
}
```
